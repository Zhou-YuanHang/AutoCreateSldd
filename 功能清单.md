## 二、Excel → M 脚本生成

**涉及文件**：`excel_to_sldd.m`（入口）、`Excel2Workspace.m`（核心函数）

### 交互功能

- [x] 自动查找当前文件夹第一个 `.xlsx`
- [x] Sheet 多选对话框（listdlg）
- [x] 默认选中 Signal 和 Parameter（按名称匹配，不写死索引）
- [x] 生成 `xxx_objects.m`（对象创建脚本）
- [x] 生成 `EnumName.m`（枚举类定义文件）

### Signal 处理

- [x] `Package.Signal` 对象创建（支持自定义包如 `myPackage.Signal`）
- [x] 存储类设置（区分默认存储类和自定义存储类）
- [x] `CoderInfo.CustomAttributes.HeaderFile`（try-catch 保护）
- [x] `CoderInfo.CustomAttributes.DefinitionFile`（try-catch 保护）
- [x] DataType / InitialValue 设置
- [x] Min / Max 设置
- [x] Description / DocUnits / Dimensions / Complexity 设置

### Parameter 处理

- [x] `Package.Parameter` 对象创建
- [x] Dimensions 在 Value 前设置（Simulink 约束）
- [x] Value 设置（Excel 列名 InitialValue → M 属性 Value）
- [x] 其余同 Signal

### Const 处理

- [x] `Simulink.Parameter` 对象创建
- [x] `CustomStorageClass` 固定为 `Define`
- [x] Value / DataType / HeaderFile 设置

### Bus 处理

- [x] `Simulink.Bus` 对象创建
- [x] Description / HeaderFile / Alignment 设置
- [x] PreserveElementDimensions（布尔值正确转换）
- [x] DataScope 设置

### BusElement 处理

- [x] 按 BusName 分组
- [x] `saveVarsTmp{1}` + `(N, 1)` 列索引格式（MATLAB 标准）
- [x] Name / Description / DataType / Dimensions / DocUnits 设置
- [x] Bus 优先于 BusElement 处理（避免名称依赖）
- [x] 未选 Bus 表时仍可正常生成

### Enum 处理

- [x] 每个 EnumName 生成独立 `.m` 文件
- [x] 格式：`classdef EnumName < DataType ... enumeration ... end`
- [x] 文件与 `xxx_objects.m` 同目录（MATLAB 自动识别枚举类）

## 四、写入数据字典 (.sldd)

**涉及文件**：`excel_to_sldd.m`（集成）、`m_to_sldd.m`（独立函数）

### SLDD 操作

- [x] 已有 SLDD → open（不删除重建，保留外部引用）
- [x] 新建 SLDD → create
- [x] 写入 Design Data 段

### 工作区变量筛选

- [x] 仅处理 Simulink.Signal / Simulink.Parameter / Simulink.Bus 及其子类
- [x] 使用 `isa` 判断继承关系（识别 `myPackage.Signal` 等）

### 条目对比逻辑

- [x] 用 `getEntry` 试错判断条目是否存在（比 `find` 可靠）
- [x] 条目不存在 → 新增（`+`）
- [x] 条目存在且值无变化 → 跳过（`-`）
- [x] 条目存在且值已变化 → 更新（`~`）
- [x] 转 `struct` 后比较（绕开 handle 类比较陷阱）

### 统计输出

- [x] 新增数量
- [x] 修改数量
- [x] 跳过（无变化）数量
- [x] 失败数量

## 七、SLDD → Excel 反向同步

**涉及文件**：`sldd_to_excel.m`（MATLAB 主逻辑）、`sldd_to_excel_helper.py`（Python 写 Excel）

### 功能

- [x] 读取 .sldd 数据字典，按类型分类（Signal / Parameter / Const / Bus / BusElement）
- [x] 扫描目录下的 Enum .m 文件，解析类定义写入 Enum sheet
- [x] 调用 Python 脚本写入 Excel（保留全部格式：颜色、字体、冻结、下拉列表）

### 写入策略

| Sheet | 第一列 | 策略 |
|-------|--------|------|
| Signal / Parameter / Const / Bus | 唯一键 | 按名匹配：同名更新、无名追加 |
| BusElement / Enum | 非唯一键 | 全量替换：清空旧数据，全部重写 |

### 与 Python 协作

- MATLAB 将数据写入临时 JSON 文件
- Python 读取 JSON，用 openpyxl 操作 Excel
- 更新已有的行在原位置（不改变顺序）
- 新增的行追加到末尾
